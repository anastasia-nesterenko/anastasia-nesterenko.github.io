Laundry Assistant

The application represents a handy database with possibility to add or edit clothes items and categories. Every item can be saved with a photo and washing, bleaching, drying and ironing instructions manually chosen from the graphical list. This application will help to store the instructions while real clothes tags are lost or their text is vanished. The application will take care of your clothes and will advise on better combinations for washing.

The application is planned to be published January, 2020. You can subscribe your email to receive an invitation when the application is published.

Closet
This screen has the list of clothes added to groups. Every item can be opened. New item can be added, existing items can be edited or removed.
Add an Item
This screen has input fields for setting item title, category and description. You can take a photo of an item by clicking on button 'Add a photo'. There is a list of option thatcan be chosen according to the instructions of the clothes.
Check the Item
This screen has all the details added to the item, including title, photo and instructions. The item can be edited.
Sign In
This screen allows to sign in for the existing users with email and password checks. For authentication, Firebase Email/Password Sign-in method is used.
Sign Up
This screen allows to sign up new users through Firebase Authentication. All fields have checks.
